第三篇游记

发表于 2017-05-02 | 分类于 南京

第三站：南京

景点： 老门东 夫子庙 江南贡院 南京大屠杀纪念馆 中山陵 总统府

美食： 南京大排档 盐水鸭 毛血旺

四月末的南京比上海要热一些，在旅馆匆匆撂下行李的我们第一站直奔老门东，
老门东游客不算多，建筑风格古朴，外加现代美感的小喷泉，风趣灵动，我们先后
品尝了南京老鸭粉，小笼包和排了很长一条长龙的烧饼，烧饼刚做好，还带着热气，
现在回味也是垂涎三尺……
南京大排档很有名，值得一去，在全国各地也有很多分店，人很多，我们去了还
等了一段时间。

小贴士
