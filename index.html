第一篇游记

发表于 2017-05-02 | 分类于 杭州

第一站:杭州

景点: 西湖（小瀛洲，湖心岛，苏堤，白堤，曲院风荷，断桥）雷峰塔 太子湾

西溪湿地 清河坊 南宋御街 吴山天风 胡雪岩故居

美食： 西湖藕粉 片儿川 东坡肉 葱包桧 特色生煎 西湖藕粉 西湖醋鱼

第一站西湖，古人说”欲把西湖比西子,淡妆浓抹总相宜”……
在春色宜人的四月来领 略西湖的韵味再恰当不过了，赶上共享单车‘风靡一时’， 我们的自助
旅行便多了一种更便捷的交通工具， 我们清晨骑着单车一路穿行在前往西湖的马路上，两边是
杭州市树香樟，这种树长得特别精致，从他分叉的规律来看，远看是一个‘丫’字，边骑车变望
向远处，一片绿色，绿的清新浓密，照相不用加滤镜，效果高清，任意一个角度都构图精巧，微
风拂面，好不惬意，一点 疲惫感都没有，完全是在享受……

第二篇游记

发表于 2017-05-02 | 分类于 上海

第二站：上海

景点： 世纪经贸 外滩 外白渡桥 豫园 城隍庙 南京东路

美食： 蟹黄灌汤包 上海特色生煎 外婆家私房菜 酸菜鱼 虾饺 粉蒸格格

第二站上海，刚落脚并没有感觉像在大城市的感觉，虽然在市中心，但是楼房不算新，感觉和沈
阳差不多。直到……
看到了东方明珠塔，上海中心（去的时候还没对游客开放），环球金融中心，世纪经贸这些巨人……
说实话，刚开始对这几个高大的建筑有些傻傻分不清楚，后来知道个头排老大的是上海中心。
听说外滩夜景漂亮，我们特意在夜幕降临时来到世纪经贸的88层，俯瞰整个上海的金融圈，特别
壮观，也特别震撼，世纪经贸下面的楼层是特别豪华的大酒店，从世纪经贸的窗外看外滩，有几艘船
在飘着，两边全是游人，城市的道路灯火通明，夜色阑珊，有点向往在这样的城市落脚，又觉得遥
不可及，觉得在这里工作的大概都是几经打拼的商业精英吧……

小贴士

  • 上海四月末微凉，记得带伞
  • 上海打车起步价14元，堵车，建议使用共享单车或地铁
  • 外滩和南京东路不管节假日还是平常人都很多，出行小心，要防止踩踏事件

第三篇游记

发表于 2017-05-02 | 分类于 南京

第三站：南京

景点： 老门东 夫子庙 江南贡院 南京大屠杀纪念馆 中山陵 总统府

美食： 南京大排档 盐水鸭 毛血旺

四月末的南京比上海要热一些，在旅馆匆匆撂下行李的我们第一站直奔老门东，
老门东游客不算多，建筑风格古朴，外加现代美感的小喷泉，风趣灵动，我们先后
品尝了南京老鸭粉，小笼包和排了很长一条长龙的烧饼，烧饼刚做好，还带着热气，
现在回味也是垂涎三尺……
南京大排档很有名，值得一去，在全国各地也有很多分店，人很多，我们去了还
等了一段时间。

小贴士

  • 南京多雨，记得带伞
  • 南京打车起步价9元，不堵车，建议使用共享单车或地铁滴滴叫车均可
  • 防止踩踏事件

第四篇游记

发表于 2017-05-02 | 分类于 扬州

第四站：扬州

景点：瘦西湖

美食：蟹粉狮子头 拆快鲢鱼头 扒烧整猪头

扬州是中国四大菜系之一淮扬菜系的发源地，菜肴注重本味、讲究火工、擅长炖焖，口味清淡鲜美、甜咸适中，
历史上有“吃在扬州”的美誉。扬州厨师犹以刀工精细著称，特别精于食材雕刻，是著名的“扬州三把刀”之一。
扬州菜以拆烩鲢鱼头、扒烧整猪头、蟹粉狮子头“三头”为代表，大煮干丝体现了淮扬菜的刀功，“三套鸭”则将菜鸽藏于野鸭腹中再将野鸭藏于家鸭腹中，野鸭喷香，菜鸽细酥，大胆…

小贴士

  • 烟花三月下扬州，三月是前往扬州游玩的最佳时间

第五篇游记

发表于 2017-05-02 | 分类于 扬州

第五站：扬州

景点：何园

何园园主何芷舠，是与晚清重臣李鸿章、光绪帝师孙家鼐联姻的扬州士绅，于光绪九年（公元1883年）
建造了这个私家园林。何园有“晚清第一园”的美誉， 园中建筑中西合璧，并由复道回廊串联，布局富
有层次感。 全园由后花园、住所院落和位于东南部的片石山房（一堆假山石，仅存遗迹，规模极小）
几部分组成，各部分之间，通过复道回廊（扬州人俗称为“串楼”，就是分为上、下两层的回廊）连成一体。
这种上、下两层的回廊，是何园建筑中的最大特色，园中人员通过回廊在园中穿行，雨天避雨、晴天遮阳，
不受天气影响。而一串串开在复道回廊上的漏窗、空窗，组成的花窗带，也是园内的一大特色。 后花园从
布局上分为东、西两个部分，主要建筑有船厅、水心亭等。东部，以“厅堂”为中心构成一组院落，其中“船厅”
最为精致，厅型似船形，四周用鹅卵石、瓦片铺地，花纹是水波状，给人以水中而居的感觉。花园的西部，
中间有一个水池，水池中建有水心亭，是园内主人观看戏剧表演的地方，“水中戏亭”的设计独具匠心。
园中的老少主人们，居住在南部的玉绣楼，这是一幢中西合璧风格的建筑。

小贴士

  • 烟花三月下扬州，三月是前往扬州游玩的最佳时间

第六篇游记

发表于 2017-05-02 | 分类于 苏州

第五站 苏州

景点：狮子林 拙政园 七里山塘

美食：生煎

一半古韵一半小资，苏州人的惬意生活是不能用文字来形容的。
有句俗话称“生在苏州、玩在杭州、食在广州、死在柳州”。既然“生在苏州”必然就有苏州寻味之处。
喝壶小茶听听昆曲是苏州人最普遍的生活方式。小桥流水人家的街景，在苏州随处可见。依着小河就
是一片商区，这里的吃喝口味偏甜，而并不像上海喜欢用浓油赤酱。
论吃，这可是渗透到了苏州人的骨子里的，最地道和最纯粹的江南点心，均有着一段“皇帝下江南”
的美丽传说。

小贴士

  • 这里的园林美景美不胜收，记得相机满格电，带上移动电源哦
  • 江南多余，记得带伞和防雨鞋套

第七篇游记

发表于 2017-05-02 | 分类于 乌镇

第七站:乌镇

景点: 东栅 西栅

乌镇，江南水乡六大古镇之一，曾名五墩和青墩，位于京杭大运河西侧。古镇虽已历经2000多年的沧桑，
但古韵犹存，东西南北四条老街呈“十”字交叉，构成双棋盘式河街平行、水陆相邻的古镇格局。乌镇，秀美
的水乡风貌、风味独特的美食佳肴、深厚的人文积淀展示出一副迷人的历史画卷。清晨在小巷走一走，细数
古镇的三十多座古桥，置身于一段古老的岁月，夜晚在河边坐一坐，小桥流水人家。那一刻，甚至有一
种错觉，仿佛乌镇等待千年就是为了等待我们的
出现。
